Technology
The 'T' in ICT refers to technology. Students in the ICT strand study various technologies, including hardware,
Hardware refers to the physical components of a computer system or any electronic device. This includes devices such as computers, servers, routers, printers, scanners, and storage devices. In the ICT field, students study hardware components, their functionalities, and configurations to understand how they interact and contribute to the overall functionality of computer systems and digital devices.
software,
Software encompasses the programs, applications, and instructions that enable computers and digital devices to perform specific tasks and functions. This includes operating systems, productivity software, programming languages, databases, and applications for various purposes such as data analysis, design, development, and communication. Students in the ICT strand learn how to use, develop, and customize software to meet specific requirements and solve problems efficiently.
networks,
Networks refer to interconnected systems of computers and devices that enable communication and data exchange. Networks can be wired or wireless and can range from local area networks (LANs) within a single location to wide area networks (WANs) spanning across geographical regions. Students learn about network architectures, protocols, security measures, and troubleshooting techniques to design, configure, and manage networks effectively in different environments.

These systems encompass a wide range of technologies, including microprocessors, memory modules, sensors, and communication interfaces. Students in the ICT strand study digital systems architecture, digital signal processing, embedded systems, and Internet of Things (IoT) technologies to understand how digital systems function and how they can be optimized and integrated into various applications and solutions.

HTML
HTML, which stands for HyperText Markup Language, is the standard language used to create and design documents on the World Wide Web. It provides the structure and layout for web pages by using a system of tags and attributes to define elements such as text, images, links, forms, and multimedia content.
CSS
CSS, or Cascading Style Sheets, is a style sheet language used to control the presentation and layout of HTML documents. It works hand in hand with HTML to define how the content of a web page is displayed, including elements such as colors, fonts, spacing, borders, and positioning.
JS
JavaScript (JS) is a high-level, interpreted programming language primarily used to add interactivity and dynamic behavior to web pages. Developed in the mid-1990s, JavaScript has become one of the core technologies of the World Wide Web, along with HTML and CSS.
PHP
PHP, which stands for Hypertext Preprocessor, is a widely-used server-side scripting language designed for web development. It is particularly well-suited for creating dynamic and interactive web pages, handling form data, managing databases, and building web applications.
SQL
SQL, which stands for Structured Query Language, is a specialized programming language used for managing and manipulating relational databases. It provides a standardized way to interact with databases, perform queries, retrieve, insert, update, and delete data, define database structures, and manage permissions and security.
JAVA
Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle Corporation) in the mid-1990s. It is designed to be platform-independent, meaning that Java programs can run on any device or platform that has a Java Virtual Machine (JVM) installed, without needing to be recompiled.
SOFTWARE ENGINEERING
Software engineering is a disciplined approach to designing, developing, testing, deploying, and maintaining software systems and applications. It applies principles, methodologies, and practices from engineering and computer science to address the challenges of building reliable, scalable, and high-quality software solutions.
